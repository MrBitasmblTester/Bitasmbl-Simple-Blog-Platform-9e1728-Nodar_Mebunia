# Bitasmbl-Simple-Blog-Platform-9e1728-Nodar_Mebunia

## Description
Build a web application where users can read and post blog articles. This project introduces basic CRUD operations, server-client interaction, and simple database handling for storing articles.

## Tech Stack
- React
- Node.js

## Requirements
- Display a list of blog articles for users to read
- Allow users to create new blog posts
- Enable updating and deleting existing blog posts
- Store blog articles in a simple backend

## Installation
- Clone: `git clone https://github.com/MrBitasmblTester/Bitasmbl-Simple-Blog-Platform-9e1728-Nodar_Mebunia.git`
- Backend: `cd backend && npm install`
- Frontend: `cd frontend && npm install`

## Usage
- Backend: `cd backend && npm start`
- Frontend: `cd frontend && npm start`

## Implementation Steps
1. Initialize Node.js backend to store blog articles.
2. Define CRUD routes for articles.
3. Set up React project.
4. Create views to list, create, update, and delete blog posts.
5. Connect React components to backend.

## API Endpoints
- `GET /articles`
- `POST /articles`
- `PUT /articles/:id`
- `DELETE /articles/:id`
